# 📱 Mobile Sales Dashboard | Power BI

A Power BI dashboard built to analyze mobile phone sales data — tracking sales performance across cities and brands to uncover key business insights.

## 🎯 Objective
This project was created as part of my data analytics portfolio to demonstrate my skills in data visualization, DAX, and dashboard design using Power BI.

## 📁  About the Data
The dataset used is **sample/practice data**, created for learning and demonstrating Power BI dashboard-building skills. It includes details like:
- Transaction ID, Date (Day/Month/Year)
- Brand & Mobile Model
- Units Sold & Price Per Unit
- Customer Name, Age & City
- Payment Method
- Customer Ratings

## 📊 Key Insights Covered
- **Sales by City** – Identifying top-performing cities by units sold
- **Sales by Brand** – Comparing performance across brands (Xiaomi, Vivo, Samsung, Apple, OnePlus)
- Payment method preferences (UPI, Cash, Credit/Debit Card)
- Customer ratings analysis across different mobile models

## 🛠 Tools & Skills Used

- **Power BI Desktop** – Data cleaning, preparation, dashboard design & visualization
- **Excel** – Data collection and initial data entry
- **DAX** – Calculated measures and KPIs
- **Data Modeling** – Structuring relationships between fields
